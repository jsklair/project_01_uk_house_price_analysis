# First-Time Buyer Affordability Pressure by Area

This page will become the GitHub Pages summary for Project 01.

## Project summary

This project analyses first-time buyer affordability pressure across areas in England and Wales, using lower-quartile house price to lower-quartile earnings ratios as the main measure.

## Why this matters

A national house price average is not enough to understand affordability. Buyers face local prices, local earnings and local constraints. This project focuses on area-level variation so that affordability pressure can be compared more realistically.

## Planned outputs
