# Notebooks

This folder contains the Jupyter notebooks used to collect, profile, clean and explore the affordability dataset.

## Notebook sequence

1. `01_data_collection_and_profiling.ipynb`  
   Records the source workbook, profiles the workbook structure and creates initial profiling outputs.

2. `02_data_cleaning.ipynb`  
   Cleans and reshapes the source data into an area-year dataset suitable for SQL, Python, Excel and Power BI analysis.

3. `03_python_exploratory_analysis.ipynb`  
   Reviews latest-year affordability pressure, regional patterns, distributions and five-year changes, then saves summary outputs and charts.

## Review notes

The notebooks are intended to read as an analytical record. Markdown cells explain the purpose of each stage, while the code cells show the reproducible steps used to create the cleaned data and exploratory outputs.
