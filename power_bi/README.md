# Power BI dashboard

This folder will contain the Power BI dashboard file and exported screenshots.

## Proposed dashboard pages

1. Executive summary
2. Affordability pressure map or regional comparison
3. Highest and lowest pressure areas
4. Change over time
5. Methodology and limitations

## Proposed measures

- Latest lower-quartile affordability ratio
- Area rank
- Change since previous year
- Change since five years earlier
- Median versus lower-quartile gap
- Pressure band

The dashboard should be designed for clarity rather than decoration. The user should immediately understand where affordability pressure is highest, how it has changed and what caveats apply.
