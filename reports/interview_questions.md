## Draft questions

### 1. Why did you use lower-quartile affordability rather than only median affordability?

Lower-quartile affordability is more relevant to the first-time buyer angle because it is closer to the entry-level housing market. Median affordability is still useful as a comparison, but it can overstate the price point faced by buyers looking at lower-cost homes.

### 2. What data quality checks did you perform?

The final answer should refer to duplicate checks, missing value checks, suspicious value checks and reconciliation between raw and cleaned row counts.

### 3. What are the limitations of the analysis?

The affordability ratio is a useful proxy, but it is not a full mortgage affordability assessment. It does not directly include interest rates, deposit size, household composition, credit criteria or parental support.

### 4. How would you improve this project with more time?

Possible improvements include adding mortgage repayment modelling, deposit saving assumptions, household income, property type mix and geospatial analysis.
